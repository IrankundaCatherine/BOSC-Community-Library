# BOSC Community Library

Welcome to the BOSC Community Library repository. This project is dedicated to developing scalable, secure communication systems in an open-source structure.

## Structure
- `.github/` contains issue and PR templates.
- `CODE_OF_CONDUCT.md` outlines community behavior standards.
- `CONTRIBUTING.md` provides detailed technical instructions for external contributors.

## How to Contribute
Please read [CONTRIBUTING.md](CONTRIBUTING.md) for details on our code of conduct, and the process for submitting pull requests to us.
